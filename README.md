# Enterprise Network Design Simulation

## Project Overview
This project simulates a corporate network using Cisco Packet Tracer. It features segmented departments (HR and Engineering) using VLANs and automated IP addressing.

## Features
- **VLAN 10 (HR):** 192.168.10.0/24
- **VLAN 20 (ENG):** 192.168.20.0/24
- **Router-on-a-Stick:** Inter-VLAN routing using a single physical interface.
- **DHCP Server:** Configured on the Router to provide automatic IP addressing.

## Tools Used
- Cisco Packet Tracer
- Cisco IOS (CLI)
